## 🆘 SE ALGO DER ERRADO

### Se Railway não reconstruiu
- Aguarde mais 5 minutos
- Verifique o painel do Railway
- Status deve estar "Running" (verde)

### Se indices não funcionam
```bash
# Verificar se foram criados
psql $DATABASE_URL -c "\d partidas"

# Se não aparecerem, executar novamente
python scripts/criar_indices.py

# Se ainda não funcionar, rodar SQL direto
psql $DATABASE_URL -c "CREATE INDEX ix_partida_data ON partidas(data);"
```

### Se site está lento
- Verificar se índices foram criados (ver acima)
- Fazer refresh da página (Ctrl+F5)
- Verificar console de erros (F12 → Console)
